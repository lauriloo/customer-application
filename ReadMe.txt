-------------------------
TO RUN APPLICATION:
-------------------------
1) Prerequisites:
Postgres with DB with a name "customer";
username & password must be configured in src/main/webapp//WEB_INF/spring.properties file to fit with DB
tables will be created by Hibernate then app is started.

2) Build application:
mvn install

3) Run application with Jetty:
mvn jetty:run

4) Open application in browser:
http://localhost:8080/

-------------------------
TEHCNOLOGIES USED:
-------------------------
* java 8
* maven 3.3.3
* Spring (MVC)
* Hibernate
* jquery(-ui)
* git
